# Print out the characteristics of the vascr data frame

Print out the characteristics of the vascr data frame

## Usage

``` r
vascr_find_metadata(data.df)
```

## Arguments

- data.df:

  the vascr data frame to interrogate

## Value

prints out the parameters of the dataframe in question

## Examples

``` r
vascr_find_metadata(growth.df)
#> 
#> Timepoints
```
