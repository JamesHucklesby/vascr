# vascr: Process Biological Impedance Sensing Data

Process complex impedance sensing datasets, including those generated by
ECIS, xCELLigence and cellZscope instruments. Data can be imported to a
standardised tidy format and then plotted. Support for conducting and
plotting the outputs of ANOVA (with appropriate tests of statistical
assumptions) and cross-correlation analysis. For data processed using
this package see Hucklesby et al. (2020)
[doi:10.3390/bios11050159](https://doi.org/10.3390/bios11050159) .
